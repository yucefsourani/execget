Simple Tool To Get Exec Command From Desktop Entry

* Requires

  * ``` pygobject3 ```
 
  * ``` python3-gobject ```



* To Use
 
  * ``` cd && git clone https://github.com/yucefsourani/execget ```

  * ``` chmod 755 ~/execget/execget ```

  * ``` ~/execget/execget```



* Install For Fedora

  * ``` sudo  dnf copr enable youssefmsourani/execget  -y ```
  
  * ``` sudo  dnf install execget  -y ```
